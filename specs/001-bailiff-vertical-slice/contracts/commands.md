# Contract ‚Äî the `bailiff` commands (slice 001)

Four verbs, stdlib `argparse`, no CLI-framework dependency. Errors print legibly
to stderr with a non-zero exit ‚Äî never a bare stack trace (FR-010).

## `bailiff discover <source> [--ref REF]`

Static, code-free inspection of one template ‚Üí JSON on stdout. No trust required.
See [`discovery-output.md`](discovery-output.md). Exit 0 on success; non-zero
`DiscoveryError` (bad YAML, no usable PEP 440 tag, missing `copier.yml`).

## `bailiff init --run-spec FILE [--check]`

Generate a project from a frozen run-spec (see [`answers-doc.md`](answers-doc.md)).

- Injects `today`, calls `run_copy(data=‚Ä¶, defaults=True, overwrite=True)`.
- `--check` runs copier's `pretend` dry run: validates inputs, **writes nothing**.
- On success (real run) writes the project, then a per-project `justfile` with the
  reproduce recipe (unless the template already ships one).
- Refuses an untrusted action-taking source (exit 3) and an unreproducible
  template (no answers-file `.jinja`, US5) before writing anything.

## `bailiff reproduce [DEST]`

Faithfully regenerate an existing project (default `DEST` = cwd).

- Requires a `.copier-answers.yml` in `DEST` (else "nothing to reproduce").
- Replays at the **recorded commit**: `run_recopy(vcs_ref=VcsRef.CURRENT,
  defaults=True, overwrite=True)` ‚Äî never a bare recopy (which would upgrade to
  the latest tag). Agent-free (Constitution III).
- Overwrites rendered files in place; leaves unrelated files untouched.

### The generated `just reproduce` recipe

`init` drops a `justfile` into the project:

```just
# Generated by bailiff ‚Äî faithful, agent-free reproduce.
# Replays the committed answers at the recorded commit (never the latest tag).
reproduce:
    bailiff reproduce .
```

## `bailiff trust add <prefix>` / `bailiff trust list`

Manage copier's `settings.yml` `trust:` list ‚Äî the ONLY writer of trust, invoked
on explicit human consent (FR-019).

- `add <prefix>` ‚Äî records the fully-expanded `https://` prefix (idempotent;
  preserves existing entries). Trust is stored expanded because copier matches the
  raw pre-expansion URL.
- `list` ‚Äî prints trusted prefixes, or `(no trusted sources)`.
- The store path honors `COPIER_SETTINGS_PATH` (used to isolate tests / CI).

## Exit codes

| Code | Meaning |
|---|---|
| 0 | success |
| 1 | a `BailiffError` (bad run-spec, copier failure, unreproducible template) |
| 2 | argparse usage error / unknown subcommand |
| 3 | `UntrustedSourceError` ‚Äî source takes actions and is not trusted |
